# README #
 
# DRAW-ER #
Draw-er is an Entity-Relationship drawer, you can work in real time with many users on the same diagram

#HOW TO INSTALL# (dev mode)
*1. npm install
*2. npm install -g bower
*3. bower install
*4. gulp
*5. open browser on 127.0.0.1:3000
